## 首页获取课程列表

### url
`/index/class`

### 请求方法
`GET`

### 请求参数
| 参数名 | 参数类型 | 说明 | 是否必须 |
| :-----: | :-----: | :-----: | :-----: |
| page | int | 分页页数，默认为1 | false |


### 响应参数
| 参数名 | 参数类型 | 说明 |
| :-----: | :-----: | :-----: |
| [].id | int | 课程ID |
| [].sold | int | 卖出数量 |
| [].price | float | 售价 |
| [].img_url | string | 图片地址 |
| [].title | string | 课程标题 |
| [].tag | string | 课程标签 |
| [].desc | string | 课程描述 |
| current_page | int | 当前页数 |
| total_page | string | 总页数 |
